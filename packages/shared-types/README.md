# Shared TypeScript Types

Shared TypeScript type definitions for the UPSC AI Platform.

## Installation

```typescript
// In other packages
import { User, Quiz, Course } from '@upsc-ai/shared-types';
```

## Types (Planned)

- **User types**: User profile, authentication
- **Quiz types**: Questions, answers, results
- **Course types**: Lessons, modules, progress
- **API types**: Request/response schemas
